# 실시간 시스템 모니터링 프로그램

Python 기반의 실시간 시스템 리소스 모니터링 도구입니다. 터미널에서 CPU, 메모리, 디스크, 네트워크 상태를 실시간으로 확인할 수 있습니다.

## 주요 기능

- **실시간 모니터링**: 1초 간격으로 시스템 정보 갱신
- **색상 표시**: 사용률에 따른 색상 구분
  - 🟢 녹색: 정상 (< 50%)
  - 🟡 노란색: 주의 (50-80%)
  - 🔴 빨간색: 경고 (> 80%)
- **진행바**: CPU 사용률을 시각적으로 표시
- **간편한 종료**: Ctrl+C로 안전하게 종료

## 모니터링 항목

1. **CPU**: 사용률 및 진행바
2. **메모리**: 사용량/전체 용량 및 사용률
3. **디스크**: 사용량/전체 용량 및 사용률
4. **네트워크**: 총 송신/수신 데이터량

## 설치 방법

### 1. Python 설치
Python 3.8 이상이 필요합니다.

```bash

```

```bash

```

```bash

```

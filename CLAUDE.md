# 실시간 시스템 모니터링 프로젝트

## 프로젝트 개요

Python 기반의 실시간 시스템 리소스 모니터링 도구입니다. 데이터 저장 없이 터미널에서 실시간으로 시스템 상태를 확인할 수 있습니다.

## skills
- **Git commit** : 사용자가 git commit 을 요청했을 때, 'skills/git_commit/skill.md' 를 참고하여 git commit 을 진행한다
 
## 핵심 요구사항

- **실시간 모니터링**: 1초 간격으로 시스템 리소스 갱신
- **저장 기능 없음**: 메모리 내에서만 동작, 데이터 저장 X
- **간단한 구조**: 경량화된 모니터링 시스템
- **터미널 기반 UI**: CLI 환경에서 동작

## 모니터링 대상

1. **CPU**: 전체 및 코어별 사용률, 빈도
2. **Memory**: 전체/사용/가용 메모리, 사용률
3. **Disk**: 전체/사용/남은 용량, 읽기/쓰기 속도
4. **Network**: 송수신 바이트, 전송 속도

## 기술 스택

- **Python 3.8+**
- **psutil**: 시스템 리소스 정보 수집
- **rich**: 터미널 UI 향상 (선택)
- **colorama**: 컬러 출력 (선택)

## 개발 가이드

### 코드 스타일
- PEP 8 준수
- 함수는 단일 책임 원칙 적용
- 명확한 변수명 사용

### 모듈화 원칙
- 각 리소스별로 독립적인 모듈
- 재사용 가능한 유틸리티 함수
- 명확한 인터페이스 정의

### UI/UX 원칙
- 직관적인 정보 표시
- 컬러 코딩으로 상태 구분
- 진행 바로 시각화

## 주요 기능

### 1. 실시간 업데이트
- 무한 루프로 지속적 모니터링
- 화면 갱신으로 최신 정보 표시

### 2. 상태 표시
- 녹색: 정상 (< 50%)
- 노란색: 주의 (50-80%)
- 빨간색: 경고 (> 80%)

### 3. 사용자 제어
- Ctrl+C로 안전한 종료
- 명령줄 옵션 지원

## 제외 사항

- 데이터베이스 연동 없음
- 파일 저장 기능 없음
- 웹 인터페이스 없음 (터미널만)
- 외부 API 호출 없음

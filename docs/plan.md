# 프로젝트 계획

## 1. 준비하기
- [x] 프로젝트 문서 작성
- [x] Python 설치 확인
- [x] psutil 라이브러리 설치
- [x] 폴더 구조 만들기

## 2. 무엇을 만들까?
CPU, Memory, Disk, Network 사용량을 실시간으로 보여주는 프로그램

### 화면에 보여줄 내용
```
============================
  시스템 모니터링
============================
CPU: 45% [█████░░░░░]
메모리: 8GB / 16GB (50%)
디스크: 100GB / 500GB
네트워크: ↑1.2GB ↓3.4GB
============================
```

## 3. 필요한 파일들
- [x] `monitor.py` - 메인 프로그램
- [x] `cpu_monitor.py` - CPU 정보 가져오기
- [x] `memory_monitor.py` - 메모리 정보 가져오기
- [x] `disk_monitor.py` - 디스크 정보 가져오기
- [x] `network_monitor.py` - 네트워크 정보 가져오기

## 4. 어떻게 만들까?

### 각 파일에서 할 일
- [x] psutil 사용해서 시스템 정보 가져오기
- [x] 정보를 보기 좋게 만들기 (GB 단위로 변환 등)
- [x] 화면에 출력하기

### 메인 프로그램 (monitor.py)
- [x] 1초마다 반복하기 (무한 루프)
- [x] 화면 지우기
- [x] 각 정보 수집하기
- [x] 화면에 표시하기
- [x] Ctrl+C로 종료하기

## 5. 색깔 넣기 (선택)
- [x] 사용률이 높으면 빨간색
- [x] 보통이면 노란색
- [x] 낮으면 녹색

## 6. 실행 방법
```bash
python src/monitor.py
```
